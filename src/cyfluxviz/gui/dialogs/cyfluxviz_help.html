CyFluxViz Workflow

Load Network for visualisation
Load Network via
File -> Import Network or File -> Open) for visualization of flux distributions within the network.

Import Flux Distributions
Flux distributions are imported for the currently selected network.
Import flux distribution in the Import Tab below.

Adapt Mapping Function
The mapping between fluxes and the network attributes can be adapted in the FluxMap Tab.

Change Sub-Network View
The NetworkView can be adapted based on the selected flux distribution to flux containing subnets (Subnet -> Flux subnet) or attribute subnets (Subnet -> Attribute subnet).

Export Images
Select Flux Distributions for export and export in PDF, SVG, EPS, JPG, PNG or BMP (Export Tab).
